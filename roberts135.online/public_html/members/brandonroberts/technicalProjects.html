Poetry Generator

Backtracking + GPT Poetry Generator

I worked on this project with a team of three other undergraduates as our final project for Professor Bergen's Fall 2022 offering of LIGN 167.

The full team consisted of Nicole Rangan, Zhengyuan Zhou, Kamen Redfield, and me.

The project GitHub repository can be found here.

Description

The poetry generator works by requesting tokens one at a time from GPT-3 based on a user-provided prompt. After each token, it checks if the result is inconsistent with the user-provided poetry format (meter, rhyme) and backtracks if there is. GPT-3 normally struggles with strict poetic forms, making frequent errors in meter and occasional errors in rhyme. This project sought to rectify this by leveraging the fact that poetic meters are deterministically verifiable, meaning it is possible to use backtracking or other search algorithms to prompt GPT-3 correct any errors it makes.

High-level pseudocode: 
generatePoem(text):
If text is a valid poem, return text
Otherwise, if text could not be the start of a valid poem, return null
Generate the top five possible tokens with GPT-3(text)
For each token:
    Append it to text
    Call generatePoem(text)
    If the result is not null, return it
    Otherwise, pop the last token

Example output

Here are the results from prompting the model to write a sonnet about linguistics:

My love for language is a thing of beauty,
A passion that has grown with each new day,
For in its depths I find a sense of duty,
To learn and teach in every single way!

The study of its forms and sounds so sweet,
The rules and structures that it does contain,
The way it changes with the times we meet,
A never ending source of joy and gain!

The beauty of its words and phrases, too,
The way they can be used to make us feel,
The power of its stories, old and new,
The way it can make our hearts so real!

My love for language is a thing of grace,
A passion that will never leave this place!

Analysis:

We can make two observations about the output:

  1. The sonnet is structurally correct
  2. The (rather subjective) quality of writing is not noticeably better than GPT-3 alone

Thus, while the project succeeded in its goal of ensuring correct form, it should be noted that it does not improve writing quality.
